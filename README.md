# AI Design System

A collection of tools and guidelines for effective AI-developer collaboration.

## Purpose

This system helps developers work with AI assistants more effectively by:

1. Providing clear, structured guidance specific to frameworks and languages
2. Respecting context window limitations
3. Enabling better planning and organization of work

## Structure

- `core/` - Universal patterns and utilities
- `frameworks/` - Framework-specific guidelines and tools
  - `nextjs/` - Next.js specific documentation and utilities
- `languages/` - Language-specific guidelines and tools
- `cursor.ts` - Position-aware code navigation and editing interface

## Usage

Include only the relevant sections when working with AI assistants to maximize context efficiency.
